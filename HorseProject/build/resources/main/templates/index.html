Welcome!

This program is a pedigree tracking database and web application that has been designed specifically for standardbred horses. The program was originally created twenty to thirty years ago by Jack Jasperson with contributions by Lawrence Cohen. The program, run entirely via the command line, has become outdated, buggy, and slow and could benefit greatly from modernization, complete with a new user interface, improved database, etc.

Functions

Horses

Add and view information about individual horses, such as breed, year of birth, sex, country of origin, gait, race type, and race records.

Go to Horses
Pedigrees

View a horse's pedigree, such as its family tree, mother, father, grandmothers, and grandfathers.
